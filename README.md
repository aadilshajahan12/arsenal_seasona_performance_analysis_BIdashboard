# ⚽ Arsenal Season Performance Analysis – Power BI Dashboard

This project is a **Power BI dashboard** created to showcase my skills in data modeling, visualization, and football analytics.  
It focuses on Arsenal FC’s season performance, structured into three analytical pages:

---

## 📊 Dashboard Pages

### 1. Season Overview
- Overall results (wins, draws, losses)
- Goals scored vs. conceded
- Home vs. away performance
- Key season KPIs

### 2. Match Analysis
- Match-by-match breakdown
- Possession, shots, xG comparison
- Tactical insights (efficiency, passing accuracy)
- Visual comparison of home vs. away stats

### 3. Player Analysis
- Individual player contributions (goals, assists, minutes, ratings)
- Starter vs. substitute impact
- Discipline metrics (cards, fouls)
- Performance trends across the season

---

## 🛠️ Tools & Skills Demonstrated
- **Power BI** → Report building and interactive dashboards
